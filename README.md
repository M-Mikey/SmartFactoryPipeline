# Smart Factory Pipeline 🏭

A real-time industrial data pipeline — foundation of an Industry 4.0 system.

## Projects
- `main` — ASP.NET Core Web API (data ingestion + Oracle DB)
- `simulator` — C# Console App (PLC machine data simulator)

## Tech Stack
- .NET 8 / ASP.NET Core Web API
- Oracle 19c + Entity Framework Core
- C# Console Simulator

## Architecture
PLC Simulator → .NET API → Oracle DB → (React Dashboard coming soon)
